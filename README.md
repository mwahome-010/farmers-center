Farmer's Center is a web application designed to help farmers identify and manage plant diseases.

The application provides educational resources about various crops, common plant diseases, and a community forum for discussions.

Objectives
----------
1. The system will allow users to login/logout.
2. The system will allow new users to register.
3. Users will be able diagnose the likely disease based on images uploaded.
4. Users will find guides on how to control and manage known pest and disease infestations.
5. Users will be able to download treatment, management and cultivation guides in form of PDF files.
6. Registered users will be able to make posts and/or comment in the forum.
7. Registered users will get reports on date joined posts and comments they made.
8. Admin will get reports on the number of registered users, unanswered forum posts and retention of users.
